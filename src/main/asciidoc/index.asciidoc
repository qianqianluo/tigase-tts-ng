Welcome to Tigase TTS-NG guide


== Overview

TTS-NG is a NextGeneration version of Tigase TestSuite intended to run automated functionality tests.
It's based on TestNG framework (similar to junit, serving as a basic test framework) and JaXMPP library (to provide client-side functionality)
